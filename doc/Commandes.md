# Documentation des commandes Git
## Commandes de base
### git init
Initialise un dépôt Git dans un répertoire.
Exemple : git init

## Commandes d’inspection
## Commandes d’inspection
### git log
Affiche l'historique des commits.
Exemple : git log
### git status
Affiche l'état des fichiers dans le répertoire de travail.
